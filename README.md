🧺 Laundry Mart

A simple and structured HTML webpage created as part of my MERN Stack course assignment from Tutedude.
This project represents a laundry service website with service details, pricing, and a booking form.

📌 Project Description

Laundry Wallah / Laundry Mart is a beginner-friendly static website built using pure HTML.
The goal of this assignment was to practice basic HTML structure, semantic elements, forms, tables, lists, and webpage layout.

This webpage provides:

A welcoming header

Introduction about the laundry service

Services offered

Price list table

Booking form

Contact information in footer

🚀 Features

Clean and structured HTML layout

About section describing the business

Image section displaying a laundry image

Services listed using an unordered list

Price table with service and price details

Booking form with required fields (Name, Email, Phone)

Footer with contact information
